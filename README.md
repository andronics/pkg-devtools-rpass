# Password Manager - Remote password storage and retrieval

**Package**: `@devtools/rpass`
**Version**: 0.1.0
**Repository**: `pkg-devtools-rpass`

## Overview

Password Manager - Remote password storage and retrieval

## Installation

### Prerequisites

This package requires the following external commands:

- `dmenu`
- `gpg`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @devtools/rpass

# Or install from local source
cd ~/.pkgs
stow rpass
```

## Usage

### Development Tool

Password Manager - Remote password storage and retrieval


## Configuration

Configuration files are typically stored in:
- `~/.config/rpass/` - User configuration
- `~/.local/share/rpass/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @devtools/rpass

# Or manual unstow
cd ~/.pkgs
stow -D rpass
```
